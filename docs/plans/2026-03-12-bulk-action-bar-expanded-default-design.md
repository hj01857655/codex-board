# 批量操作栏常驻展开设计

**日期：** 2026-03-12
**范围：** BulkActionBar 交互简化

## 目标
- 批量操作栏默认展示全部按钮。
- 移除“更多操作/展开/收起”入口，减少无意义操作。
- 保持现有批量逻辑与文案一致。

## 现状
- 通过 `expanded` 状态控制折叠与展开。
- 折叠态显示“测试选中项 + 更多操作”。
- 展开态显示完整按钮集合。

## 方案
- 删除 `expanded` 相关状态与副作用。
- 移除折叠态 UI 区块和“更多操作/展开/收起”按钮。
- 保留完整按钮区作为唯一渲染结构。

## 影响范围
- `src/components/bulk/BulkActionBar.tsx`

## 风险与回退
- 风险低：仅影响展示与交互，不触及业务逻辑。
- 回退方式：恢复 `expanded` 状态与折叠 UI 分支。

## 验收
- 选中任意条目后，批量栏直接显示完整按钮。
- 不再出现“更多操作”“展开/收起”。
- 批量测试/启用/禁用/删除/清空行为与结果不变。
